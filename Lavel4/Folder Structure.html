Folder Structure

  1. Root Directory  :Main folder containing all website files.

  2. HTML Files :Store main .html files at the root lavel for easy access.

  3. CSS Folder :Creat a css/ folder for all Cascading Style Sheets.

  4. JS Folder :Use a script/ folder for Javascript files.

  5. Imeges Folder  :Store images in an images/ or images folder.

  6. Assets :Other assets like fonts can go in an assets/ folder.

  7. Sub-directories  :For malti-page website, use folders to categorize content
